DOCTYPE

Primeira tag de um arquivo html, servindo para especificar o tipo do documento e qual vesão do html se estã utilizando.

Tag Html

Tag que encapsula todo o documento html.

Tag Head

Local aonde as tag que comunicão com o browser oque será utlizado na construção do site e alguma informações para os mecanismos de pesquisa.

Tag Body

Tag aonde o conteudo do site se apresenta.

Tag a:

A tag 'a' é utlizada para fazer a conecção de algum endereço local ou na web atraves do atributo href Exemplo: Google.

Caminhos(links)

Absoluto

Quando fazemos referencias a um caminho de uma pagina na web atraves de uma URL, como no exemplo da tag 'a'.

Relativos

Quando estamos trabalhando com arquivos locais e queremos navegar entre eles, exemplo: Clique aqui.

Background

Atributo de tag que muda a cor do Background.

Box Model

Caixa onde recebem o conteudo e criam funcionalidas de controlar a altura, largura, padding, bordas e etc.

Exemplo de box model

Utilizando a tag div e estilizando no css definimos varios atribulos, sempre deixe a margin do body em 0.

Estilos do browser

Os browsers definem algumas estilizações padrões para algumas tags, isso afeta as heranças que alguns atributos podem ter, isso pode causar alguns problemas de tag não recebem do a herança, contudo pode ser resolvido colocando ao lado da tag de estilo do css inherit.

Display

Existem estilos de escrito tanto na escrita a mão quanto na web, por exemplo escrevemos da esquerda para direita, de cima para baixo, contudo os japoneses escrevem de cima para baixo da esquerda para direita.

Elementos inline e de bloco, os inline serão escritos em uma linha sem a quebra de linha, logo conseguem ficar um do lado do outro, sem respeitas os valores de width, height e etc, no caso dos elementos de bloco isso não acontece pois tem uma quebra de linha no final deles.

Display none remove o elemento da tela.

Display inline-block faz com que o elemento inline respeite os atributos de elemento block.

Imagens

Tag que insere imagens é a img, que se fecha nela mesma, nas imagens se pode colocar width 100% no css para ela ficar flexivel ao width, max-width delimitará que a imagem vai ocupar 100% da tela até o sei tamanho maximo, e colocar como display: block para ela se comportar como elemento de bloco.

Svg arquivo de imagem que é definido por codigo vetorial.
