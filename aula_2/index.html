DOCTYPE

Primeira tag de um arquivo html, servindo para especificar o tipo do documento e qual vesão do html se estã utilizando.

Tag Html

Tag que encapsula todo o documento html.

Tag Head

Local aonde as tag que comunicão com o browser oque será utlizado na construção do site e alguma informações para os mecanismos de pesquisa.

Tag Body

Tag aonde o conteudo do site se apresenta.

Tag a:

A tag 'a' é utlizada para fazer a conecção de algum endereço local ou na web atraves do atributo href Exemplo: Google.

Caminhos(links)

Absoluto

Quando fazemos referencias a um caminho de uma pagina na web atraves de uma URL, como no exemplo da tag 'a'.

Relativos

Quando estamos trabalhando com arquivos locais e queremos navegar entre eles, exemplo: Clique aqui.
